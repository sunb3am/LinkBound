Set Up Your First Session

A Session maps to one LinkedIn account. Each session keeps a separate browser profile so cookies, login state, and rate limits are fully isolated.

LinkBound — Help & Guide

🚀 Campaigns

Upload an Apollo/ZoomInfo CSV (or paste LinkedIn profile URLs) → choose an action and message → preview the batch → launch. Always use Dry Run first to validate without sending.

  • Auto — Smartest option: sends a connect request if not connected, or a DM if already 1st degree.
  • Connect + Note — Connection invite with up to 300 characters.
  • Connect only — Silent connection with no note.
  • Message — Direct message to 1st-degree connections only.

📋 Templates

Create reusable message templates. Use {first_name}, {company}, {sender} as dynamic variables. Templates can be selected per campaign and tracked individually in Analytics.

▶ Live Run

Monitor the active run in real-time. The activity feed shows each profile as it is processed. Use the Pause / Resume and Stop Run buttons in the top bar to control execution.

👥 Audience Manager

Every contact you've ever reached out to is stored here. Filter, sort, search, and export to CSV. The dedup engine prevents reaching out to the same person twice across batches.

📊 Analytics

Track total contacts, sends per day (30-day trend), and template usage. Refresh the dashboard after each campaign to see updated numbers.

⚙️ Sessions (Operators)

Each Session is one LinkedIn account. Sessions are fully isolated: separate Chrome profile, separate dedup scope, separate rate limits. Create a session per person or per LinkedIn account you manage.

  • On first use, a Chrome window opens so you can log in and complete 2FA.
  • After login, the profile is saved — future runs start without a login prompt.
  • Deleting a session removes it from the dropdown but does not delete its browser profile or data.

🔒 Safety Limits

Configure daily cap, min/max delay between actions, and business-hours constraints in Settings → Safety Limits. The runner enforces these automatically and stops when the cap is hit.

✨ AI Features

Configure a Gemini API key in .env to enable AI-powered message generation, improvement, and per-profile personalisation. The AI pill in the top bar shows active / inactive status.

📂 CSV Column Mapping

LinkBound auto-detects columns: linkedin_url, first_name, last_name, company, email. If your CSV uses different names, the smart heuristic matcher still finds them. You can also paste raw LinkedIn URLs one per line.
